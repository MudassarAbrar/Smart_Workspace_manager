# Smart Workspace Manager

## Project Guide
This is a comprehensive workspace management system built with C++ and Qt.

## Team Roles
- **MEMBER 1**: Core Logic (Person, Employee, Manager, Department)
- **MEMBER 2**: Scheduling & Requests (Project, Task, Meeting, LeaveRequest)
- **MEMBER 3**: Control, UI, Reports (User, WorkspaceManager, main)

## Build Steps
1. Create build directory: `mkdir build && cd build`
2. Run CMake: `cmake ..`
3. Build: `cmake --build .`
4. Run: `./SmartWorkspaceManager` (or `SmartWorkspaceManager.exe` on Windows)
